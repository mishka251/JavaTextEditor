# JavaTextEditor
Простой текстовый редактор на Java
Задача: "оживить" кнопки
1. [x] Копировать 
2. [x] Вставить
3. [x] Цвет
4. [ ] Статистика(показ формы без данных пока)
5. [x]  Enter
6. [x] Сохранить
7. [x] Загрузить 
8. [x] Найти
9. [x] Показать буфер
10. [x] Выравнивание справа
11. [x] Выравнивание слева
12. [x] Выход
-----
13. [x] Показ текущей  статистики
